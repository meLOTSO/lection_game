# Lection Game (pyglet)

Небольшая учебная игра на Python + `pyglet` с простым менеджером состояний (меню / игра / диалог / пазл / победа / поражение).

## Требования

- Python 3.10+ (подойдёт и новее)
- `pyglet`

Установка зависимости:

```bash
pip install pyglet
```

## Запуск

Из корня проекта:

```bash
python main.py
```

Параметры окна и FPS находятся в `data/init.py`.

## Управление

Точные подсказки по управлению для каждого состояния лежат в `models/control_info.py` и соответствующих файлах `controls/types/*_controls.py`.

Базовые клавиши:

- **В игре (`RUNNING`)**: A / D — движение, Space — прыжок, E — взаимодействие
- **В диалоге (`DIALOG`)**: Enter — следующая реплика
- **В пазле (`PUZZLE`)**: Enter — подтвердить, Backspace — удалить символ, Delete — очистить, Tab — закрыть
- **В меню (`MENU`)**: Enter — начать

## Архитектура (кратко)

- **`states/`**: перечисление `GameState` и текущее состояние.
- **`contexts/`**:
  - `state_controls_pair.py` — связывает состояние и обработчики ввода
  - `state_draws_pair.py` — связывает состояние и функции отрисовки
  - `state_context.py` — переключение состояния (обновляет и ввод, и отрисовку)
- **`controls/`**: роутинг событий окна (`on_key_press`, `on_text`, `update`) в активный набор контролов.
- **`draws/`**: отрисовка конкретных экранов/оверлеев.
- **`models/`**: игровые модели (диалог, сценарии, инвентарь, платформы и т.п.).
- **`game_rules/`**: “правила”/логика, которая реагирует на события и меняет состояние игры.
